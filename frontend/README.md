FRD Fund Frontend

How to run:

1. cd frontend
2. npm install
3. npm run dev

Place the existing logo at `c:\frdfund\images\logo-frdfund.png`. Vite serves `/images` from workspace root so the app will load it at `/images/logo-frdfund.png`.

Theme:
- Use the Theme controls on the login page to adjust `Primary` and `Accent` colors. Values can be hex colors like `#155a6a` or CSS color names.
